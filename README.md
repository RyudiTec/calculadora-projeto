Este projeto da calculadora foi desenvolvido usando HTML, CSS e JavaScript, com base em conteúdos aprendidos em um vídeo no YouTube. O objetivo é simular o funcionamento de uma calculadora básica, realizando operações como soma, subtração, multiplicação e divisão.

No HTML, criei a estrutura da calculadora com um visor para mostrar os números e vários botões para os valores e operações.
No CSS, organizei os botões em formato de grade e estilizei a interface para ficar parecida com uma calculadora real.
No JavaScript, aprendi a capturar os cliques nos botões, mostrar os valores no visor, realizar os cálculos e exibir o resultado.

Além disso, o código permite limpar o visor e corrigir valores digitados. Esse projeto me ajudou a praticar lógica de programação, eventos de clique e manipulação do DOM com JavaScript.
